Leurs - Discord Bot

A feature-rich Discord bot designed to enhance your server experience with economy, moderation, music integration, and utility commands.

Add to Discord Learn More
Discord Bot Python Open Source

Features

Economy System

  • Virtual currency management
  • Betting and gambling mechanics
  • Job market with career paths
  • Daily rewards and income

Advanced Moderation

  • User management (ban, kick, mute)
  • Warning system with history
  • Smart message deletion
  • Role and nickname management

Last.fm Integration

  • Real-time music tracking
  • Server-wide now playing
  • User music statistics
  • Scrobble counts and history

Leveling System

  • XP-based progression
  • Server leaderboards
  • Visual progress bars
  • Level-based rewards

Administrative Tools

  • Automated role restoration
  • Message caching and search
  • Custom embed creation
  • Reaction roles

Utility Commands

  • Birthday tracking
  • Timezone management
  • Custom quote generation
  • Information commands
